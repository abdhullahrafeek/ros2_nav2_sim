# My Robot ROS2 Project

This repository contains a ROS2 project for simulating and controlling a robot with SLAM and navigation capabilities in Gazebo and RViz.

---

## Project Requirements

- **Operating System:** Ubuntu 22.04  
- **ROS2 Distro:** Humble  
- **Dependencies:**  
  - `slam_toolbox`  
  - `nav2` (Navigation2)  
  - `gazebo_ros_pkgs`  

> Make sure you have installed ROS2 Humble before setting up this workspace.
